# the-data-guide
This repository contains my structured notes on data analysis, data science and related topics. 
 
They're organized as notebook guides that are meant to document a progression toward proficiency in any of these areas.
The goal is to create a clear reference that benefits both my own development and others following a similar path.

Here's some of the principles I'll be following when writing these:
- **Fundamentals First**: The first few notebooks should always cover the building blocks of whatever is to come.
- **Clear Progression**: Each notebook should build on the previous ones. Advanced topics should feel familiar and not overwhelming.
- **Informal, But Informative**: None of this should feel like a chore to read. But no information should be inaccurate, and no notebook shouldn't be useful.
- **Constant Revision**: Notebooks should be revisited and updated to make sure they're still useful and accurate.

This repository evolves alongside my growth in the field.
